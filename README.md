\# IoT Device Security Monitor \& Hardening Tool



Automated security platform for discovering, assessing, and securing IoT devices on networks.



\## Features

\- Network discovery and device fingerprinting

\- Vulnerability assessment (passwords, ports, firmware, CVEs)

\- Automated remediation

\- Web dashboard for monitoring

\- Continuous security monitoring



\## Tech Stack

\- Backend: Python 3.x

\- Database: PostgreSQL

\- API: Flask/FastAPI

\- Frontend: React + TypeScript

\- Scanning: Nmap, Scapy
